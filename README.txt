Добрый день, перед вами реализация для сбора рецензий о фильме "Побег из Шоушенка".
Это вторая реализация, только уже при помощи неоф. API. Для работы программы вам необходимо зарегистрироваться
на сайте https://kinopoiskapiunofficial.tech/ и получить Api ключ, далее переименовать application.properties.origin
в application.properties и там внести ваш API ключ, после этого программа заработает.
